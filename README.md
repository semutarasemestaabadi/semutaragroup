# PT Semutara Semesta Abadi — Landing Page

Landing page bilingual (EN/ID) untuk **PT Semutara Semesta Abadi**, perusahaan marketing dan commodity trading Indonesia yang menghubungkan produk lokal — khususnya pemanis alami seperti gula semut, gula aren, dan gula cair singkong — dengan pasar global.

## ✨ Fitur

- **Bilingual EN/ID** — toggle bahasa disimpan di localStorage
- **Sticky Header** — transparan saat di atas, solid saat scroll
- **Smooth Scroll** — navigasi halus dengan active section highlight
- **Animate on Scroll** — animasi fade-in pada elemen
- **WhatsApp Floating Button** — akses cepat chat
- **Scroll to Top** — tombol kembali ke atas
- **Fully Responsive** — desktop, tablet, mobile

## 🗺️ Site Map

| Section | ID | Deskripsi |
|---|---|---|
| Hero | `#hero` | Banner utama + CTA |
| About Us | `#about` | Profil perusahaan |
| Vision | `#vision` | Visi perusahaan |
| Mission | `#mission` | 5 misi utama (grid card) |
| Products | `#products` | Gula Semut, Gula Aren, Gula Cair Singkong |
| Contact | `#contact` | Alamat, Email, Follow Us (social media standalone) |
| Features Bar | — | 4 keunggulan (natural, UMKM, global, quality) |
| Footer | `#footer` | Info perusahaan, quick links, kontak |

## 🛠️ Tech Stack

- **HTML5** — semantic, section-based structure
- **CSS3** — vanilla, desktop-first, CSS Grid + Flexbox, CSS Variables, animations
- **JavaScript** — vanilla, no framework
